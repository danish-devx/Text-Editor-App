# 📝 Text Editor App

A simple and interactive **Text Editor Web Application** built using **HTML, CSS, and JavaScript**.  
This project allows users to write and format text directly in the browser with multiple styling options.

🔗 Live Demo  
https://danish-devx.github.io/Text-Editor-App/

---

## 🚀 Features

- Bold text
- Italic text
- Underline text
- Increase / Decrease font size
- Text alignment (Left, Center, Right)
- Text color picker
- Background color picker
- Uppercase / Lowercase text transform
- Copy text to clipboard
- Reset editor
- Modern **Neumorphism UI design**

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- SweetAlert2

## 🎯 Purpose of This Project

This project was built to practice **DOM manipulation, event handling, and UI styling** using JavaScript and CSS.

It demonstrates how a basic **rich text editor interface** can be implemented in the browser using simple web technologies.
